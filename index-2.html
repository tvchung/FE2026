📚 Giới thiệu HTML

HTML là gì?

HTML (HyperText Markup Language) - Ngôn ngữ đánh dấu siêu văn bản là một ngôn ngữ dùng để tạo nên các trang web. Nó không phải là ngôn ngữ lập trình mà là ngôn ngữ đánh dấu (markup language).

💡 Định nghĩa: HTML là một tập hợp các thẻ (tags) được dùng để định dạng và cấu trúc nội dung trên trang web.

Đặc điểm của HTML

✓ Ngôn ngữ đánh dấu: Sử dụng các thẻ để bao quanh nội dung

✓ Dễ học: Cú pháp đơn giản, không yêu cầu kỹ năng lập trình nâng cao

✓ Độc lập với nền tảng: Chạy trên mọi hệ điều hành và trình duyệt

✓ Kết hợp với CSS và JavaScript: HTML cung cấp cấu trúc, CSS cung cấp style, JavaScript cung cấp tương tác

HTML vs HTML5

HTML HTML5
Phiên bản cũ hơn Phiên bản mới nhất (ra mắt 2014)
Thẻ cơ bản như <div>, <span> Thẻ ngữ nghĩa <header>, <nav>, <article>
Hỗ trợ multimedia yếu Hỗ trợ <video>, <audio>, Canvas
Không có API nâng cao API phong phú: Geolocation, Storage, WebSocket

🏗️ Cấu trúc của Trang HTML

Bộ khung cơ bản của một trang HTML

Mỗi trang HTML đều có một cấu trúc cơ bản bao gồm các phần chính:

<!DOCTYPE html> <html> <head> <!-- Thông tin về trang web --> </head> <body> <!-- Nội dung trang web --> </body> </html>

Giải thích các phần chính

<!DOCTYPE html> - Khai báo kiểu tài liệu

Thẻ khai báo (declaration) cho biết trình duyệt rằng đây là một tài liệu HTML5. Phải đặt ở dòng đầu tiên.

<!DOCTYPE html>

<html>...</html> - Phần tử gốc

Là phần tử cha của toàn bộ tài liệu HTML. Tất cả các phần tử khác nằm bên trong nó.

<html lang="vi"> <!-- Nội dung --> </html>

Thuộc tính lang: Chỉ định ngôn ngữ của trang (vi = tiếng Việt, en = tiếng Anh)

<head>...</head> - Thông tin meta của trang

Chứa các thông tin về trang (metadata) như tiêu đề, bộ ký tự, stylesheet, script, ... Nội dung trong thẻ này không hiển thị trên trang.

<head> <meta charset="UTF-8"> <title>Tiêu đề trang</title> </head>

<body>...</body> - Nội dung trang

Chứa tất cả nội dung hiển thị trên trình duyệt như text, hình ảnh, liên kết, form, ...

<body> <h1>Tiêu đề chính</h1> <p>Đây là nội dung trang</p> </body>

🎯 Thẻ Xác định Thông tin cho Trang Web <head>

Nhiệm vụ của thẻ <head>

Thẻ <head> chứa các thông tin metadata của trang web như:

  • Tiêu đề trang
  • Bộ ký tự (charset)
  • Viewport cho responsive design
  • Liên kết đến stylesheet CSS
  • Script JavaScript
  • Thông tin cho search engine
  • Icon của trang

Các thẻ con thường dùng trong <head>

<meta charset="UTF-8"> - Khai báo bộ ký tự

Cho biết trình duyệt sử dụng bộ ký tự Unicode (UTF-8) để hiển thị text, bao gồm các ký tự đặc biệt, tiếng Việt, ...

<meta charset="UTF-8">

<meta name="viewport"> - Cài đặt viewport

Giúp trang web hiển thị đúng trên các thiết bị di động. Rất quan trọng cho responsive design.

<meta name="viewport" content="width=device-width, initial-scale=1.0">

Giải thích:

  • width=device-width - Chiều rộng = chiều rộng thiết bị
  • initial-scale=1.0 - Zoom ban đầu = 1 (100%)

<link rel="stylesheet" href="style.css"> - Liên kết stylesheet

Dùng để liên kết file CSS bên ngoài để trang web có style đẹp.

<link rel="stylesheet" href="style.css">

<script src="script.js"></script> - Liên kết file JavaScript

Dùng để nhúng file JavaScript để trang web có tương tác và logic.

<script src="script.js"></script>
⚠️ Lưu ý: Các thẻ script để load file bên ngoài thường đặt ở cuối thẻ <body> (thay vì trong <head>) để trang web load nhanh hơn.

🏷️ Thẻ Tiêu đề <title> và Thẻ Meta <meta>

Thẻ <title> - Tiêu đề Trang

Thẻ <title> xác định tiêu đề của trang web. Tiêu đề này sẽ hiển thị trên:

  • Tab trình duyệt
  • Bookmark
  • Kết quả tìm kiếm (Search Engine)
  • Lịch sử duyệt web

Ví dụ Thẻ <title>

<head> <title>Trang Chủ - DevMaster.edu.vn</title> </head>

Kết quả: Tab trình duyệt sẽ hiển thị "Trang Chủ - DevMaster.edu.vn"

Thẻ <meta> - Siêu Dữ liệu (Metadata)

Thẻ <meta> cung cấp thông tin (metadata) về tài liệu HTML. Có nhiều loại meta tag khác nhau:

<meta name="description"> - Mô tả Trang

Cung cấp mô tả ngắn gọn về nội dung trang. Được hiển thị trong kết quả tìm kiếm.

<meta name="description" content="Học lập trình web từ cơ bản đến nâng cao">

<meta name="keywords"> - Từ Khóa

Danh sách các từ khóa mô tả nội dung trang, giúp SEO.

<meta name="keywords" content="HTML, CSS, JavaScript, Web Development">

<meta name="author"> - Tác Giả

Xác định tác giả của trang web.

<meta name="author" content="Trần Văn A">

<meta name="theme-color"> - Màu Chủ đề (HTML5)

Xác định màu chủ đề của trình duyệt (chủ yếu trên mobile).

<meta name="theme-color" content="#667eea">

Ví dụ Hoàn chỉnh cho <head>

<head> <meta charset="UTF-8"> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <title>Học HTML - DevMaster.edu.vn</title> <meta name="description" content="Hướng dẫn học HTML5 từ cơ bản đến nâng cao"> <meta name="keywords" content="HTML, HTML5, Web Development"> <meta name="author" content="DevMaster"> <meta name="theme-color" content="#667eea"> <link rel="stylesheet" href="style.css"> </head>

📝 Thẻ Xác định Phần Thân (Nội dung) Trang Web <body>

Thẻ <body> là gì?

Thẻ <body> xác định phần thân (body) của tài liệu HTML. Tất cả nội dung hiển thị trên trình duyệt đều nằm bên trong thẻ này.

💡 Quy tắc: Một tài liệu HTML chỉ có một thẻ <body> duy nhất.

Nội dung bên trong <body>

Thẻ <body> có thể chứa các phần tử sau:

📰 <h1> - <h6>

Tiêu đề (Headings)

📄 <p>

Đoạn văn (Paragraph)

🖼️ <img>

Hình ảnh (Image)

🔗 <a>

Liên kết (Link)

📋 <ul>, <ol>, <li>

Danh sách (Lists)

📊 <table>

Bảng (Table)

📝 <form>

Biểu mẫu (Form)

📦 <div>

Khối (Container)

Ví dụ Thẻ <body>

<body> <header> <h1>Chào mừng đến với trang web</h1> </header> <nav> <a href="#">Trang chủ</a> <a href="#">Về chúng tôi</a> <a href="#">Liên hệ</a> </nav> <main> <section> <h2>Tiêu đề chính</h2> <p>Nội dung trang web...</p> </section> </main> <footer> <p>© 2026 Bản quyền</p> </footer> </body>

🎯 Một số Thẻ Cơ bản trong HTML, HTML5

Thẻ Tiêu đề (Heading Tags)

<h1> đến <h6> - Tiêu đề

Định nghĩa các tiêu đề, với h1 là lớn nhất, h6 là nhỏ nhất. Mỗi trang nên có một h1 duy nhất.

<h1>Tiêu đề cấp 1</h1> <h2>Tiêu đề cấp 2</h2> <h3>Tiêu đề cấp 3</h3>

Thẻ Đoạn và Định dạng (Paragraph & Formatting)

<p> - Đoạn

Định nghĩa một đoạn văn bản.

<p>Đây là một đoạn văn bản.</p>

<strong> - Văn bản quan trọng (Bold)

Tạo văn bản đậm với ý nghĩa ngữ nghĩa quan trọng.

<p>Đây là <strong>văn bản quan trọng</strong></p>

<em> - Văn bản nhấn mạnh (Italic)

Tạo văn bản nghiêng với ý nghĩa nhấn mạnh.

<p>Đây là <em>văn bản nhấn mạnh</em></p>

<br> - Xuống dòng

Tạo một dòng mới. Đây là thẻ tự đóng (self-closing tag).

<p>Dòng 1<br>Dòng 2</p>

<hr> - Đường kẻ ngang

Tạo một đường kẻ ngang để tách biệt nội dung.

<p>Phần 1</p> <hr> <p>Phần 2</p>

Thẻ Danh sách (List Tags)

<ul>, <li> - Danh sách không thứ tự

Tạo danh sách với bullet points.

<ul> <li>Mục 1</li> <li>Mục 2</li> <li>Mục 3</li> </ul>

<ol>, <li> - Danh sách có thứ tự

Tạo danh sách với số hoặc ký tự.

<ol> <li>Bước 1</li> <li>Bước 2</li> <li>Bước 3</li> </ol>

Thẻ Media (Media Tags) - HTML5

<img> - Hình ảnh

Chèn hình ảnh vào trang.

<img src="hinhanh.jpg" alt="Mô tả hình ảnh">

<video> - Video - HTML5

Nhúng file video vào trang (không cần plugin Flash).

<video width="320" height="240" controls> <source src="video.mp4" type="video/mp4"> Trình duyệt không hỗ trợ video </video>

<audio> - Âm thanh - HTML5

Nhúng file âm thanh vào trang.

<audio controls> <source src="nhac.mp3" type="audio/mpeg"> Trình duyệt không hỗ trợ audio </audio>

Thẻ Semantic - HTML5

<header>, <nav>, <main>, <section>, <article>, <aside>, <footer>

Các thẻ ngữ nghĩa giúp mô tả rõ ràng cấu trúc trang web và cải thiện SEO.

<header>...</header> <nav>...</nav> <main> <section>...</section> <article>...</article> </main> <aside>...</aside> <footer>...</footer>

Thẻ Biểu mẫu (Form Tags)

<form>, <input>, <label>, <button>

Các thẻ để tạo biểu mẫu nhập liệu.

<form> <label for="name">Tên:</label> <input type="text" id="name" name="name"> <label for="email">Email:</label> <input type="email" id="email" name="email"> <button type="submit">Gửi</button> </form>

Lưu ý Quan trọng

⚠️ HTML5 đơn giản hóa cú pháp:
  • Không cần viết XHTML đóng nghiêm ngặt
  • Thẻ tự đóng như <img>, <br>, <hr> không cần />
  • Thuộc tính không cần dấu ngoặc kép nếu không có khoảng trắng
  • Nhưng vẫn nên tuân theo chuẩn để code sạch và dễ bảo trì

✅ Tóm Tắt

HTML và HTML5 là nền tảng cơ bản của web development. Những kiến thức chính cần nắm vững:

📄 Cấu trúc

<!DOCTYPE>, <html>, <head>, <body>

🏷️ Thẻ Meta

charset, viewport, title, description, keywords

📝 Nội dung

Heading, Paragraph, List, Image, Link

🆕 HTML5

Semantic tags, Video, Audio, Canvas, Storage

🎓 Bước tiếp theo: Sau khi nắm vững HTML, hãy học CSS để style trang web, rồi JavaScript để tạo tương tác.
